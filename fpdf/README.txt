Important : Ce projet utilise la librairie FPDF pour la generation de PDF.

Comme le telechargement automatique a echoue, vous devez installer FPDF manuellement :

1. Allez sur http://www.fpdf.org/
2. Cliquez sur "Telechargement" (Download).
3. Telechargez le fichier ZIP.
4. Extrayez le contenu.
5. Copiez le fichier 'fpdf.php' (et le dossier 'font' si present) dans ce dossier 'fpdf/'.

Le fichier final doit etre : C:\xampp\htdocs\project web\fpdf\fpdf.php
